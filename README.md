# IoT-Based Battery Monitoring System

## About

This project presents an **IoT-based Battery Monitoring System** designed to efficiently monitor lead-acid batteries in vehicles. Unlike traditional Battery Management Systems (BMS) that provide limited information through basic LED indicators, this system offers a comprehensive and interactive monitoring solution through a **Mobile App and Website Interface**.

---

## 🚀 Features

- Real-time monitoring of battery parameters:
  - **State of Charge (SoC)**
  - **Voltage**
  - **Temperature**
  - **Charging / Discharging Status**
- Wireless communication using **Bluetooth** and **WiFi**
- Supports both electric and non-electric vehicles
- Mobile App interface (developed using Android Studio)
- Web Interface for easy access and monitoring

---

## 🛠️ Technologies Used

- **Hardware:**
  - Arduino UNO
  - Bluetooth Module
  - WiFi Module
  - Temperature Sensor
  - 9V Lead-Acid Battery
  - Power Supply Adapter

- **Software:**
  - Arduino IDE
  - Android Studio
  - Sublime Text 3 (for Web Development)

---

## ✅ Advantages

- Low-cost implementation
- Easy monitoring of battery parameters
- User-friendly interfaces
- High efficiency in sensing battery data

---

## ⚠️ Limitations

- Corrosion may damage battery terminals
- Potential for short-circuits in IoT device
- Requires basic knowledge of internet and smartphone use

---

## 🌟 Future Enhancements

- Deploy system to the cloud for remote access
- Enable advanced data analytics for battery health management
- Integrate predictive maintenance features
